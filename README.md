# 2027 특성화고 대학 입시결과 데이터베이스

2027학년도 특성화고 특별전형 상담을 위해 대학별·학과별 입시결과를 수집하고 정규화한 프로젝트입니다. 학생의 내신 등급과 희망 계열을 기준으로 대학을 탐색하고, 안정·적정·소신 지원 후보를 비교할 수 있는 웹 화면을 포함합니다.

## 현재 데이터 범위

- 입시결과 레코드: 1,808건
- 결과가 확보된 대학: 61개교
- 등급 데이터가 확보된 대학: 40개교
- 대학·학과 조합: 1,167개
- 등급 비교 가능 레코드: 642건

대학이 소수 모집단의 결과를 비공개하거나 공식 결과표에 특성화고 전형을 별도로 싣지 않은 경우가 있어, 대학별 확보 범위는 `data/coverage_report.json`과 `data/target_coverage.csv`에서 확인할 수 있습니다.

## 구조

```text
.
├─ 2027_특성화고_모집대학.xlsm  # 조사 대상 대학 원본 목록
├─ collect_nesin_pdfs.py         # nesin 입시결과 PDF 수집기
├─ pipeline/                     # 추출·정제·DB 생성·커버리지 점검
├─ data/
│  ├─ admissions.sqlite          # 상담용 정규화 데이터베이스
│  ├─ DATA_DICTIONARY.md         # 테이블·지표 설명
│  └─ *.json, *.csv              # 출처 및 커버리지 메타데이터
└─ web/                          # 상담용 Next.js 웹
```

## 웹 실행

Node.js 20 이상이 필요합니다.

```bash
cd web
npm ci
npm run dev
```

테스트는 다음 명령으로 실행합니다.

```bash
cd web
npm test
```

## 데이터베이스

핵심 테이블은 `institutions`, `documents`, `admission_tracks`, `programs`, `results`, `metrics`이며, 상담 화면에서 바로 조회할 수 있도록 `counseling_results`와 `latest_counseling_results` 뷰를 제공합니다. 자세한 필드 정의는 [데이터 사전](data/DATA_DICTIONARY.md)을 참고하세요.

원본 PDF와 대학별 원본 엑셀은 용량·재배포 범위를 고려해 Git 저장소에 포함하지 않습니다. 수집기와 출처 레지스트리를 이용해 다시 확보할 수 있으며, 정제된 결과와 출처 URL은 DB 및 메타데이터 파일에 포함되어 있습니다.

## 주의사항

이 데이터는 입시 상담을 돕기 위한 참고 자료입니다. 대학별 환산 방식, 모집 인원, 공개 기준이 서로 다르므로 실제 지원 전에는 각 대학의 최신 모집요강과 공식 입시결과를 반드시 확인해야 합니다.
